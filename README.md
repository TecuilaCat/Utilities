## Maven-Dependency
Add the following dependendy to the `pom.xml` file
```xml
<dependency>
    <groupId>com.github.tecuilacat</groupId>
    <artifactId>Utilities</artifactId>
    <version>1.0.1</version>
</dependency>
```
